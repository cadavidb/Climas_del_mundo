# Título del Proyecto

Climas del mundo

### Pre-requisitos 📋

debes tener instalado git en tu computadora

## Comenzando 🚀

git clone https://github.com/cadavidb/Climas_del_mundo.git




### Instalación 🔧

abre una terminal en el proyecto y escribe 

npm install


### sigue con 
node app -d *ciudad*



## Construido con 🛠️

node js y la api open weather map
